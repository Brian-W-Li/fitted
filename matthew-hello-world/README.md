# Matthew Li - Hello World Demo

React Native (Expo) app for the Lab1 assignment
Uses the Expo "Tabs" template

## Tech Stack
* **Framework:** React Native
* **Tooling:** Expo Go

## How to Run This App

### Prerequisites
* Node.js installed on your computer
* Expo Go app instsalled on iOS/Android phone

### Installation
1. Navigate to the project directory:
   ```bash
   cd matthew-hello-world
   ```

## Get started

2. Install dependencies

   ```bash
   npm install
   ```

3. Start the app

   ```bash
   npx expo start
   ```
