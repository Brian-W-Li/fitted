# Fitted — User Manual (Draft)

> This is a draft user manual for **Fitted**
>
> Last updated: **March 11, 2026**

## Product purpose
Fitted is a web app that allows you to store items from your wardrobe and get outfit recomendations based on what you own and your likes and dislikes for outfits.

## Intended user audience
- People who want to save time choosing an outfit (e.g., students and working professionals).
- Users who want outfit ideas using only their own closet items.
- Users who want to personalize their outfit suggestions
