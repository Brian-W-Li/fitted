## Project Ideas & Interests

An app that helps users decide what to wear using items from their own wardrobe. Many people struggle with daily outfit decisions and spend a lot of time on that.

Key ideas:
- Allowing users to upload their clothing items
- Use computer vision to identify key points about the item (like type, color, fit, etc)
- Automatically generating complete outfits by mixing and matching pieces
- Letting users favorite items or rate outfits so the app learns preferences

For the tech-stack:
- Next.js for the frontend
- Python or Node.js for the backend
- PostgreSQL vs NoSQL database
- Computer Vision for recognizing clothing items
- ML for personalized outfit suggestions
