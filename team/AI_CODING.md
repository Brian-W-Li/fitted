## Coding with AI

#### Matthew
AI tools used: Cursor, ChatGPT

Outcomes:
I used Cursor to help write and fix code more quickly, such as suggesting fixes and autocompleting code. I used ChatGPT to explain errors or design choices, help debug issues, and clarify concepts when I was confused.

Usefulness:
Cursor was helpful for speeding up coding and reducing repetitive work. I also used it to find and fix bugs efficiently. ChatGPT was helpful for understanding problems and thinking through solutions. Going forward, I would use Cursor for writing code faster and ChatGPT for explaining. 

Ensuring correctness and understanding:
I tested the code in my project instead of just trusting it. I also read through the AI’s suggestions and asked ChatGPT to explain things when I didn’t understand. I made sure to modify AI-generated code to fit our project instead of copying it directly.
